GitHub Pages Deployment
Host for free — no server required
Step 1

Prepare the repo

Push all HTML, JS, WASM, and INI files to your GitHub repo. Add Data.rsdk via Git LFS (it's 38 MB — too large for a regular commit).

Step 2

Enable Pages

Go to Settings → Pages, set Source to main branch, root folder. GitHub will give you a https://username.github.io/repo/ URL.

Step 3

Add COOP headers

Create _headers (Netlify) or a GitHub Actions workflow to set Cross-Origin-Opener-Policy: same-origin and Cross-Origin-Embedder-Policy: require-corp for full SharedArrayBuffer support.

Size tip for GitHub free tier (1 GB LFS limit):
Compress Data.rsdk with zstd -19 Data.rsdk (typically ~24 MB) and decompress it in web.html using the DecompressionStream API. The WASM files are already tiny (~3 MB each). You can safely exclude SonLVLObjDefs/, SDL2.dll, and icon/*.pdb to trim the repo further.
